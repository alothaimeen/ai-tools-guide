# AI Tools Guide

موقع تفاعلي يعرض أدوات AI بدون كود مع ميزات:
- 🔍 بحث ديناميكي
- 📊 مقارنة أدوات
- ❤️ إضافة للمفضلة وتقييم محلي
- 📤 مشاركة عبر تويتر
- 🌙 الوضع المظلم مع حفظ التفضيل

## طريقة النشر

1. قم بإنشاء مستودع باسم `ai-tools-guide`.
2. ارفع الملفات الأربع أعلاه إلى GitHub.
3. فعّل GitHub Pages من إعدادات المستودع.
4. ستحصل على رابط مباشر للموقع.
